# Code of Conduct

This project follows version 2.1 of the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), the standard code of conduct adopted by thousands of open source projects.

Please read the full text at the link above. In short: contribute constructively, assume good faith, communicate with respect, and report any concerns to the maintainer at **nicholas@nicholaslawrence.studio**.

The maintainer reviews reports promptly and will take whatever action is appropriate to maintain a productive, welcoming space for contribution.
